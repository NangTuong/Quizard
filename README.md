# Quizard

## Purpose 
Allows users to take quizzes generated by authorized users and create an account to make their own quizzes to challenge other users knowledge

## Built With
* HTML
* CSS
* JavaScript
* React
* Node.js
* GraphQl
* Apollo
* MongoDB
* Heroku

## Website
https://ltb-quizard.herokuapp.com/
